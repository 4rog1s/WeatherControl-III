BASIC CONTROL
Move mouse to change perspective.
Press W to move forward, S to move backward, A to move left and D to move right.
Use sliders or input numbers to change the effects.
Press Save to save, and Load to load an extra save file.
Save files will be storage in Assets/Resources folder.
The half-transparency Canvases is for Fog of War effect, which to block some views around the character.
